# Lethe Custom Boss Guide

A step-by-step guide to creating custom boss encounter mods for **Limbus Company** using the **Lethe** modding framework.

Uses the **Yellow Harpoon** (노란작살) Vespa boss mod as a running example throughout.

---

## Guide Structure

| Chapter | Content |
|---------|---------|
| [Chapter 1](chapter1_EN.md) | Finding your target character's data in dumpedData — appearance, skills, motions, and ID collision |
| [Chapter 2](chapter2_EN.md) | Building a full encounter mod — stage registration, boss unit, skills, buffs, and patterns |
| [Chapter 3](chapter3_EN.md) | Advanced Lua with ModularSkillScripts — runtime animation control and conditional logic |

---

## Requirements

- LetheLauncher with the Lethe framework installed
- A text editor (VS Code recommended)
- Basic JSON knowledge
